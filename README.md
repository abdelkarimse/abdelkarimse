## 🤖 AI Agent Architecture

<div align="center">

```
┌─────────────────────────────────────────────────────────┐
│                   🧠 Agent Orchestration                │
│                                                         │
│   ┌──────────────┐         ┌──────────────────────┐    │
│   │  LangGraph   │◄───────►│  State Machine       │    │
│   │  Workflows   │         │  (Nodes + Edges)     │    │
│   └──────┬───────┘         └──────────────────────┘    │
│          │                                              │
│   ┌──────▼───────┐         ┌──────────────────────┐    │
│   │  LangChain   │◄───────►│  Tools / RAG / APIs  │    │
│   │  Chains      │         │  Memory / Vector DB  │    │
│   └──────┬───────┘         └──────────────────────┘    │
│          │                                              │
│   ┌──────▼───────┐         ┌──────────────────────┐    │
│   │  LLM Layer   │◄───────►│  OpenAI / HuggingFace│    │
│   │  (Backbone)  │         │  Fine-tuned Models   │    │
│   └──────────────┘         └──────────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

</div>

I design and implement **multi-agent systems** using LangGraph's stateful graph execution model — enabling agents to reason, plan, use tools, and collaborate autonomously. From RAG pipelines to agentic loops with memory and reflection, I build AI that doesn't just respond — it *acts*.

## 🛡️ Security & Quality Pipeline

<div align="center">

```
  Code Push ──► SonarQube Analysis ──► Snyk Vulnerability Scan
                      │                          │
               Code Smells &              Dependency CVEs &
               Coverage Gates            License Compliance
                      │                          │
                      └──────────► CI Gate ◄─────┘
                                       │
                              Docker Build & Deploy
```

</div>

Every project I ship goes through a **quality + security gate**: static analysis with **SonarQube** (coverage, code smells, duplication), dependency scanning with **Snyk** (CVEs, license issues), and containerized delivery with **Docker**. Clean code isn't optional — it's the baseline.
